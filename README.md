## 👨‍💻 About me

`TypeScript` 기반으로 안정적이고 확장 가능한 프론트엔드를 만드는 개발자입니다.

React와 Vue로 실제 서비스를 운영하며, 유지보수하기 쉬운 코드와 아키텍처에 집중합니다.

의료·업무 시스템 개발 경험을 통해 실사용자 관점의 실용적인 문제 해결을 추구합니다.

프론트엔드를 중심으로 백엔드, 인프라, 모바일까지 전체 개발 프로세스를 이해하고,

팀의 개발 경험 향상과 명확한 코드 컨벤션을 중요하게 생각합니다.
